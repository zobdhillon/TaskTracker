# TaskTracker

A Laravel-based task management app with a full authentication system.

## Features

- User authentication (login, register, password reset, email verification)
- Task management (create, edit, delete)
- Categories for organizing tasks
- Recurring tasks with frequency options (daily, weekly, monthly)
- Dashboard with task stats
- Dark mode UI built with Tailwind CSS

## Tech Stack

- **Backend:** Laravel, PHP
- **Frontend:** Blade, Tailwind CSS, JavaScript
- **Build Tool:** Vite
- **Database:** MySQL
